## 제목학원

![Untitled](%E1%84%89%E1%85%A5%E1%84%87%E1%85%B5%E1%84%89%E1%85%B3%20%E1%84%89%E1%85%A9%E1%84%80%E1%85%A2%F0%9F%98%86%2083035b6d43424ce091d4fc7f657909e2/Untitled%2011.png)

<aside>

```
📌 “제목학원”은 매일 새롭게 새로운 짤로 게시됩니다. 유저분들은 제목학원 페이지에서  오늘 진행중인 제목학원과 지난 제목학원을 확인 하실 수 있습니다.
```

</aside>

- **오늘 진행중인 제목학원**
    - 해당 제목학원을 클릭하면 댓글을 달 수 있는 상세 페이지로 이동합니다.
    - 현재 좋아요 1등인 댓글 정보를 간략하게 확인 할 수 있습니다.

- **역대 제목학원**
    - 지난 날들의 제목학원 기록을 확인 할 수 있습니다.


</aside>

- **인기순/최신순/과거순으로 댓글을 조회할 수 있습니다.**
- **마음에 드는 댓글에 좋아요를 누를 수 있습니다.**
    - 하나의 댓글에는 한번만 좋아요를 누를 수 있습니다.
- **마음에 드는 댓글에 대댓글을 달아 반응을 나타내실 수 있습니다.**
- **제목학원은 매일 아침 7~ 저녁10시에 진행되며**
    
    **종료되는 시점에 좋아요 1,2,3등 유저분들께 수상을 해드립니다.**
    
- **지난 제목학원은 조회는 가능하시만 좋아요나 댓글은 작성하실 수 없습니다.**
